# Credit Card Default Prediction

This project builds a classification model to predict credit card defaults using a dataset of 30,000 account records from a Taiwanese bank. In this context, a false positive occurs when the model incorrectly flags a reliable customer (resulting in a small administrative review cost), while a false negative occurs when the model misses a customer who actually defaults (resulting in a severe financial loss for the balance). Because missing a default is vastly more expensive than a false alarm, the ultimate goal of the project is to evaluate and select the model that minimizes the total expected financial cost for the bank.
